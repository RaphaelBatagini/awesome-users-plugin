## Interacting with plugin code

1. Checking code standards:
```sh
$ vendor/bin/phpcs --standard="Inpsyde" <path>
```
Where <path> is at least one file or directory to check, e.g.:
```sh
$ vendor/bin/phpcs --standard="Inpsyde" ./src/ ./my-plugin.php
```
